## 项目目标

实现一个功能与 Wistia 视频播放器相似的视频播放器。


## 使用 Video.js 实现视频播放器

我们将借助 [Video.js](https://videojs.com/) 项目来实现与 Wistia 外观和功能相似的视频播放器。
